FROMULARIOS

Ideas principales

  • form: Define un formulario que se utiliza para recopilar información.

  • input: Es donde el ususario coloca la información.

  • type: Es el que le da sus caracteristicas al formulario.

Apuntes

Los formularios e las páginas web son utilizados para recolectar o resgitrar información que va directamente a un servidor.

Estos formularios nos permiten hacer elementos muy utiles como lo pueden ser; recolección de datos, inicios de sesión, encuestas, etc.

Resumen

Los formularios son una parte fundamental para ciertos tipos de páginas web ya que nos ayudan a tener un contacto directo con el usuario.

COLORES

Para seleccionar un color en CSS existen ditintos metodos;

  • Hexadecimal; Trabaja con 16 caracteres. El 0 representa que no hay color y la F el tono en su maxima intensidad.
  • RGB y RGBA; Representa 3 colores (Red, Green, Blue) y a partir de las mezclas de estos se forman nuevos colores.
  • HSL; Hue Saturation Luminosidad o Matiz saturación y luminosidad es medida a partir de grados (H), y en porcentajes (L y S).

Dentro de una página web es recomendable no utilizar más de 3 colores para sus elementos ya que puede saturar e incomodar la visión del usuario

Es recomendable utilizar colores en armonía, ya sea utilizando contrastes, degradados o lo monocromático.

Para CSS existe distintos métodos para seleccionar colores, entre ellos encontramos los modelos; RGB, CMYK, HSL, HWB, hexadecimales.

Resumen

En CSS podemos seleccionar colores de distintas maneras con distintos tipos de medición, ya sea midiendolos a partir de caracteres, porcentajes, grados, etc.

Apuntes

CSS3 es un intento de reducir el uso de codigo de "Javascript" y para estandarizar funcioes populares, CSS3 no solo cubre diseño y estilos web sino tambien forma y movimiento.

Gracias a estas propiedades agregadas podemos generar esquinas redondeadas a elementos, sombras paralelas e internas, transformaciones de elementos y reposicionamiento de distinos elementos.

Resumen

Las propiedades de CSS3 nos ayudan a utilizar funiones esteticas y funcionales sin la necesidad de utilizar javascript.

ESPECIFICIDAD

Ideas principales

Orden jerarquico de las etiquetas:

  1. Estilos inline

  2. ID's

  3. Clases, atributos y pseudo-clases

  4. Elementos y pseudo-elementos

Apuntes

La especificidad es el medio por el cual los navegadores deciden qué valores de propiedad CSS son los más relevantes para un elemento y, por lo tanto, se aplicarán.

La especificidad se aplica de manera jerarquica, dando preferencia a ciertos tipos de elementos. Cuando varias declaraciones cuentan con la misma especificidad, la última declaración en el CSS es aplicada.

Resumen

Según sea el orden jerarquico, se les da prioridad a ciertas declaraciones. En este tema encontramos; clases, atributos, pseudo-clases, elementos y pseudo-elementos.

HEADER

Ideas principales

Elementos que pueden incluirse en el header:

  • Logotipo

  • Datos de contacto

  • Redes sociales

  • Tienda online

  • Botones destacados

  • Barras fijas

  • Selector de idiomas

  • Formulario de suscripción

Apuntes

El header es una de las partes más importantes en una página web, ya que es el primer elemento que visualiza un usuario al entrar a la web

El header debe ser un espacio genérico, es decir, que sea igual esteticamente en todas las paginas del sitio web.

Dentro del header podemos colocar distintos elementos útiles como lo son; los datos de contacto, una tienda online, redes sociales, etc.

Resumen

El header es un espacio genérico el cual debe mantenerse igual en toas las páginas de un sitio, este puede llevar distintos elementos muy útiles.

FOOTER

Ideas principales

  • Textos legales

  • Cookies

  • Redes sociales

  • Copyright/créditos

  • Datos de contacto

  • Sellos de confianza

  • Call To Action

  • Formulario de suscripción

Apuntes

El footer o pie de página es un espacio genérico el cual debe mantenerse igual en todas las páginas de un sitio.

Si es bien aprovechado, el footer puede establecer elementos muy útiles, como contener formularios de suscripción, datos de contacto y lo más importante que son los avisos legales.

Resumen

El footer es una parte fundamental de todo sitio web, ya que este cuenta con elementos muy importantes como lo son los textos legales y los sellos de confianza. Además, es un elemento genérico al igual que el header, por lo que se mantiene igual en todas las páginas del sitio.
